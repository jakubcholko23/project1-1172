Bookstore Database Schema
For this project, I have worked with 2 other students to create a bookstore database. We started by defining out entities, atttributes and relationships that will be needed for the database and created a ER diagram. Then we created our final draft with an EER Diagram and used MySQl to start creating the database. We programmed the database in MySQL and added data to see if it worked and simulated search results by using SQL commands such as joins. We also normalized the schema so that there we no redundandcies within it.
